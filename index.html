모바일 설치 페이지

DCLiveChat를 Tampermonkey에 설치하세요

모바일 브라우저에서 아래 버튼을 누르면 `dclivechat.user.js` 설치 화면으로 바로 이동합니다. 설치 뒤에는 DCInside나 FMKorea 페이지를 다시 열면 됩니다.

Tampermonkey에 설치하기 GitHub 저장소 열기
  • Tampermonkey가 연결된 모바일 브라우저에서 여는 것이 가장 편합니다.
  • 설치용 스크립트는 최신 `min.js`를 불러와 실행합니다.
  • 버튼이 바로 설치로 이어지지 않으면 아래 직접 주소를 열어도 됩니다.
직접 주소: https://goisbyhi.github.io/dclivechat/dclivechat.user.js
