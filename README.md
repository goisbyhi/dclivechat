# DCLiveChat

모바일에서도 바로 설치할 수 있도록 Tampermonkey용 사용자 스크립트 링크를 추가했습니다.

- 설치 페이지: [https://goisbyhi.github.io/dclivechat/](https://goisbyhi.github.io/dclivechat/)
- 직접 설치 링크: [https://goisbyhi.github.io/dclivechat/dclivechat.user.js](https://goisbyhi.github.io/dclivechat/dclivechat.user.js)

## 설명

- `dclivechat.user.js`는 모바일 Tampermonkey 설치용 파일입니다.
- 설치 뒤에는 최신 `min.js`를 불러와 실행합니다.
- 대상 사이트는 `gall.dcinside.com`, `m.dcinside.com`, `www.fmkorea.com`, `m.fmkorea.com`입니다.
